# 自分って何やってて何ができるんだろう
ふと思いました。「自分って何やってて何ができるんだろう」って。

私は、小学校高学年の頃に父からPCを触らせてもらってたり、環境を用意してもらったりなど、非常に積極的に技術に触れさせてくれていました。その影響だと思いますが、比較的早い段階から技術に対する興味がありました。

Minecraftなどのゲームに始まり、ゆっくり実況などのYoutubeに関係する動画・画像編集技術、インターネットを使った情報収集、ハードウェアに関する知識など、様々な分野、ジャンルにおいて広く浅くではありますが接触してきました。

で、現在の2026年。小学校のときから見たらだいぶ年月が経ちましたが、自分の根本は変わっていないと思います。それを前提とし、今自分が何をしていて、何ができるのかというのを改めて整理してみようと思いました。

## 昔と今の違い
この場合、**昔**というのは小学校・中学校の時代を指すと定義させてください。

昔は今のようなWeb系技術ではなく、ゲーム・動画などのいわゆる”クリエイター”に興味を持っていました。ここで話すのは少し恥ずかしい内容な気もしますが、半分黒歴史みたいな動画も過去のYoutubeアカウントにアーカイブが残っていました。

当時は今のようにAIもなく、Google検索でも豊富でたくさんの情報が”整理”されている状態ではすぐに見つかりませんでした。そんな中、AviutlやYMMなど、動画編集ソフトを頑張って構築・整備して使える状態に持っていき、エンコード・アップロードができたときはすごく嬉しかったのだと思います。

中学に入るとまあMinecraft意外にも色んなゲームをし始めました。FortniteとかApexとか。その時期にリリースされたFPSは学校で流行っていた印象があります。（荒野行動とかね）

こう振り返ってみると、昔、つまり小中の時代は動画クリエイターとして色々活動することに興味が大きく、見ていた動画、配信などに非常に大きな影響を受けていたのだと考えられます。

自分の昔と今、**何が大きく違うのでしょうか？**

---

私の回答は”何も変わっていない”です。

昔も今も、**”関わってくれる人たちを楽しませたい”** という気持ちは変わっていません。振り返ってみて実感しました。

動画クリエイター？配信者？ゲーム開発者？インフラエンジニア？

みんな同じです。同じ志を持っているといっても過言ではないと思います。
ユーザー、プレイヤー、リスナー、チームメイト、友達。
規模感なんて関係ありません。その人達が少しでも面白いと思ったり、便利だと思って使ってくれたり。

とにかく、お互いに関わり合えていることに大きな価値があり、そこに楽しさ、面白さがあるのだと思います。

もうサービス終了してしまいましたが、私は４〜５年ほどMinecraftサーバーの運営・開発を行っていました。
アクティブユーザー数自体は2桁どころか多くて5人くらいの規模でしたが、そのユーザーの方々にできるだけ楽しんでもらえるようにサーバー・コミュニティ運営を心がけていましたし、当時から”インフラ”への意識を持っていました。

一時期、ゲーム配信を行っていたときもありました。リアルタイムにリスナーと繋がり、交流をし、場合によっては一緒にゲームを楽しむ。すごく好きな時間でした。今でもたまにやっています。

## 過去の経験から何が得られたのか

### 1. 技術を愛せる自分になれた

まず断言できることがあります。

それは、「**技術を愛せる自分になれたこと**」です。

ここまでの自分を構築してくれたのは、親とか友達とか、それは当然ですが、私は自分1人で行動することが多かったため、そのモチベーションを保ってくれた要因は間違いなく**技術**です。

私の性格にはなるのかもしれませんが、物事の仕組みとか関係するなにかを調べるのが昔から好きで、技術の技術を探ることも多々ありました。

例として、一番関わりが深いMinecraftを挙げましょう。

- 3D
	- モデリング
	- テクスチャ
	- グラフィック
- ゲーム
	- サンドボックス
	- マルチプレイヤー
		- Minecraftサーバー
- Java
	- プログラミング言語
	- JVM

とりあえず適当に羅列するだけでもこんなにたくさんあります。振り分けはかなり大雑把ですが、とにかく、技術にはたくさんの技術が連なっていることは見て取れると思います。

〇〇やってて楽しい、気になる --> じゃあ〇〇の□□ってどうやって動いてるんだろう？

を無限に繰り返せばあら不思議、技術が大好きな人間が出来上がりました。

### 2. 幅広い技術に興味を持てた

先程の項目に近い内容にはなりますが、２つめは「**幅広い技術に興味を持てた**」ことです。

うまくできるできないは置いておいて、今私が”一応できる”レベルでざっと書くとこうなります。

- 動画編集 (長尺も可、カット・テロップ)
- 画像編集 (Photoshop、フリーソフト、サムネイル作成)
- 写真・動画撮影 (ゲーム含む、OBS、その他キャプチャ技術等)
- 音楽制作 (DTM、簡単な打ち込み)
- 3Dモデリング (BlockBench, Blender)
- パソコン
	- 自作PC (BTO買ったことない)
	- 自宅ネットワーク
- AIツール
	- ローカルLLM
	- クラウド
- フロントエンド (HTML, CSS, JS, TS)
- バックエンド (Python, FastAPI)
- インフラ
	- オンプレ (ProxmoxVE, Ubuntu Server, Docker, Kubernetes)
	- クラウド (Cloudflare, AWS, GCP)
	- 簡易的なネットワーク
- コミュニティ運営

まだある気もしますが、とりあえず出てきたのはこれくらいです。

少し理解できる、くらいの感覚ですが、なんだかんだ全部使う場面あるので役に立ってます。
プログラマなんだったら別に動画編集とか画像編集なんてあんまり関係ないじゃん、と思われる方も一定数いると思いますが、本当にそんなことはないです。

実例を出してみます。
例えば動画・画像編集の場合。
フロントエンドやアプリケーションそのもののデザイン、ゲームデザインなど、ユーザーや開発者が”目で見る”コンテンツは当然、デザインへの意識が必要です。編集の経験が少しでもあると、レイアウトやフォントなど、細かい部分までもこだわりを持つ事ができます。

別に、こうだからみんな動画編集をしよう、みたいな話ではなく、今触ってるアプリの**技術**は必ず自分の役に立つ場面が来るんだろう、と心の奥底でも良いので覚えておいてくれると嬉しいです。

### 3. 技術に対する抵抗感が０

０は結構過言な可能性ありますが、新しい技術に対する抵抗感はほんとになくなりました。
難しそうと感じることはあっても、絶対やりたくないな。みたいなのはないです。

というより、過去に避けた技術はだいたい触ることになってる気がするので、機会がもし訪れたら、先回りして使ってみる、というのも面白いかもしれませんね。

抵抗感がない・少ないことは技術者としても、日常生活でもすごく大切だと思います。
特に現代においてはAIを使うことでなんでも検索でき、なんでも解決できるようになってきましたが、AIツールなしで自分でなにかをしようとなった際、Google検索やWikipediaみたいなのに抵抗感があったら大変じゃないですか！？

まあ、これは言いすぎですが、抵抗感がないともしその時が来たときにチャレンジしやすいので、先程の話と繋がりますが、今後自分のなにかの役に立つかもしれません。

## 今の自分を完全に形成することになった転換点

2024年7月。今の自分を”完全”に形成することになったとあるコンテンツに出会いました。

アイコンからお察しの方もいらっしゃるかもしれませんが、「VRChat」です。

私自身、VRデバイス自体には昔から興味があり、累積で初代Quest ~ Quest 3, Quest Pro, XR Eliteで５台の所有歴があり、当然、VRChatについても認知はしていました。

正直、コミュニケーションを得意としているタイプではなかったので、勇気も出ずなかなか踏み込んでいなかったのですが、いわゆる”スタンミブーム”が始まりそうなくらいの時期に流行に乗ろうということでVRChatの世界に突入しました。

---

VRChatについて知らない方のために念のため説明しておきますと、VRChatはデスクトップ・VR・モバイルなど様々なプラットフォームでプレイが可能な、メタバースと呼ばれるジャンルの１つのゲームタイトルです。

Unityで開発されており、ユーザーはある程度のランクを持つとアバター・ワールドをUnityでビルドし、VRChatのサービス上にアップロードすることで他のユーザーもそれを閲覧したり遊んだりすることができます。

今の時代多様性多様性と言われますが、個人的にVRChatは多様性の塊なのである意味理想的なのではないか？と思っています。

---

### VRChatの可能性を知る

ここ掘り下げると凄い文章量になるかもしれないので省略しますが、VRChatでインターネットの新次元を感じました。場所関係なく友人とそこにいるかのように交流することができ、かつゲームまでできてしまうと。ボイスチャットとは別世界の交流の仕方です。

アバターやワールドの改変次第では、「魔法のようなものを魔法のように」、「リアルなものをリアルに」基本的にすべてがクリエイターの思いのままであり、自由なサンドボックスなのです。

様々な趣味、仕事、交流のスタイル、VRChatの楽しみ方。みんな違ってみんな良いとはまさにこのこと。VRChatは現実との境目が自由自在に変わっていて面白いです。私はその部分に大きく可能性を感じています。

# 今後どうしていきたいのか

私は、**コンテンツやサービスを支える、インフラ**そのものを支えるエンジニアになることを将来の目標としています。

SNS、ゲーム、Webアプリケーション、様々なサービス、その種類は問いません。世の人々が利用しているサービスを支え、発展させ、ユーザーを笑顔にできたらそれで十分です。簡単ではないと思いますが、そうできるように頑張ります。

そして、ここまではキャリアとしての目標。

ここからが今回語りたかった、コアの部分かもしれません。

---

できれば今年から実行に移したいですが、私のもう一つの目標。それは「VRとリアルの融合、可能性を推進すること」です。

まずいいます。僕はVRが好きです。VRChatという世界が好きです。
でも、当然リアルも大好きです。自然、都市、人々の声、実際に自分の感覚でないと感じられないものが多くあります。

どちらも好きなのでどうしようとなるところですが、じゃあどっちも混ぜてしまえば良いと。

仮想空間上でリアルなコミュニケーションができるようになれば、場所・時間の縛りがかなりなくなります。目標を読むとすごく壮大に感じるかもしれませんが、僕ができることといえば、イベントを主催したり、SNSを活用してVR空間・メタバースと呼ばれるものがあるということをより多くの人達に知ってもらうことくらいです。

私意外にもVRChatに大きな影響を受け、色々な方向で進んでいった人たちがたくさん居ます。進む先がどうであれ、それはその人の人生です。運命というのがもし定められているのであれば、そのとおりに動いているのかもしれませんし、それは予測不可能です。

場所に囚われず、多種多様な人々と交流を持てるプラットフォーム、使い方はその人次第だよ、というのを伝えつつ、もっとたくさんの人に知り、利用してほしいと願います。
